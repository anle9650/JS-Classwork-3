# Google Book Search

This is a web application that allows a user to search Google Books by various keywords, including title, author, and publisher. It displays the titles and
information returned for each search. The user can navigate between pages to obtain and display the next set of results.

## Usage

Enter in the name of a title, author, or publisher in the search bar, and click search. Use the next button to display the next 20 results, or previous to return
to the previous 20 results.
